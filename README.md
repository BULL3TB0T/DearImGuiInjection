# DearImGuiInjection
Inject [Dear ImGui](https://github.com/ocornut/imgui) into any unity game.
## Renderers
- DirectX 11
- DirectX 12
- Vulkan
- OpenGLES2
- OpenGLES3
- OpenGLCore
